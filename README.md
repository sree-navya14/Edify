# Edify 🎓

Edify is a full-stack educational platform designed to make learning more accessible through free courses, scholarships, mentorship opportunities, career guidance, and AI-powered support.

🔗 **Live Demo:** https://edify-frontend.web.app/

🔗 **Repository:** https://github.com/sree-navya14/Edify

---

## 🚀 Overview

Edify aims to bridge the gap between learners and educational opportunities by providing a centralized platform where students can:

* Enroll in free courses
* Track learning progress
* Earn course completion certificates
* Discover scholarship opportunities
* Connect with mentors
* Receive career guidance
* Access AI-powered assistance

The platform focuses on making quality education accessible while helping students build skills and advance their careers.

## ⚙️ Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/sree-navya14/Edify.git
cd Edify
```

### 2. Install Dependencies

```bash
npm install
```

### 3. Configure Environment Variables

Create a `.env` file and add:

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
```

### 4. Start the Application

```bash
npm start
```

### 5. Open in Browser

```text
http://localhost:3000
```

## 🔐 Security
